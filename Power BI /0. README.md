# BI-проекты (Power BI)

## 1. Дашборд воронок и динамики посетителей (Яндекс.Метрика)

**Цель:**  
Обеспечить команду простым и наглядным инструментом для мониторинга поведения пользователей на сайте в режиме реального времени, без необходимости работы со сложным интерфейсом Яндекс.Метрики.

**Задача:**  
Создать дашборд, который автоматически обновляет данные и показывает:
- ключевые воронки (регистрация, авторизация, изучение продукции, покупка) с детализацией по этапам и % дохода до цели
- динамику числа посетителей month-over-month (MoM)
- фильтрацию по месяцам и типу устройства

**Решение:**  
- Написан Python-скрипт для выгрузки данных через API Яндекс.Метрики
- Настроено автоматическое обновление данных в Power BI
- Построены воронки с расчётом конверсии на каждом этапе
- Добавлена динамика MoM для визуализации трендов
- Реализованы фильтры для гибкого анализа по периодам и устройствам

**Результат:**  
Команда получила интуитивно понятный дашборд с актуальными данными в реальном времени. Воронки и динамика MoM стали доступны без обращения к Яндекс.Метрике, что ускорило принятие решений.

**Стек:** Python (pandas, requests), Яндекс.Метрика API, Power BI

## 2. Дашборд план-факт анализа рекламных кампаний (4 квартал 2024)

**Цель:**  
Оценить выполнение плановых показателей по рекламным кампаниям за 4 квартал 2024 года и выявить причины отклонений.

**Задача:**  
Построить дашборд, который:
- показывает фактические vs плановые значения по ключевым метрикам (клики, охват, показы, CTR, бюджет, CPC)
- позволяет фильтровать данные по платформе, дате, источнику, кампании+креативу
- отображает динамику метрик во времени
- содержит общую таблицу с данными и наглядные графики
- формирует автоматические выводы о выполнении плана

**Решение:**  
- Данные из файлов плана и факта объединены по месяцам и ключам (кампания, креатив)
- Рассчитан фактический бюджет = клики × CPC
- Построены графики динамики по каждой метрике для платформ OLV и retargeting
- Реализована логика автоматического вывода: «план выполнен» / «не выполнен» на основе сравнения факта с планом
- Дашборд состоит из 6 страниц:
  - главная (общие показатели)
  - детализация по кликам, охватам, показам, бюджету, CTR
- По каждой метрике сформулированы гипотезы о причинах отклонений (например, завышенные планы, узкий таргетинг, технические ограничения платформы)

**Результат:**  
Руководство и команды получили инструмент для оперативного контроля выполнения плана и поиска точек роста. Выявлено, что основная проблема — невыполнение плана по охватам и показам при перевыполнении CTR, что указывает на возможные ошибки в планировании или настройках таргетинга.

**Стек:** Power BI (DAX, Power Query), Excel (исходные данные)

## 3. Отчет по спецпроекту (кастомная аналитика через Яндекс.Метрика API Logs)

**Цель:**  
Провести анализ эффективности спецпроекта, включая охваты, поведение пользователей на лендинге, вовлечённость в опросы и KPI коммуникаций (email, WhatsApp, приложение).

**Задача:**  
- Выгрузить данные о поведении пользователей через Яндекс.Метрика  
- Рассчитать ключевые метрики (охваты, визиты, вовлеченность, конверсии)  
- Построить воронки конверсии  
- Визуализировать отчет в едином стиле 

**Решение:**  
- Написан Python-скрипт для выгрузки логов через Яндекс.Метрика API Logs
- Данные обработаны и агрегированы в pandas  
- Построены кастомные воронки с использованием `matplotlib.pyplot` и `matplotlib.patches.Polygon`  
- Цвета воронок адаптированы под брендбук клиента  
- Создан структурированный PDF-отчет с разделами: охваты, лендинг, опросы, вовлечённость, KPI рассылок, методология

**Результат:**  
Заказчик получил детализированный отчет с инсайтами:  
- выявлены источники с наибольшей вовлеченностью  
- построены воронки опросов для разных аудиторий  
- рассчитано выполнение плана по ключевым метрикам (охваты, переходы, вовлеченность)  
- предложены рекомендации по улучшению коммуникаций

**Стек:**  
- Python (pandas, requests, matplotlib, matplotlib.patches.Polygon)  
- Яндекс.Метрика API Logs
- Power BI (сборка отчета)  
- PDF (финальная презентация)
